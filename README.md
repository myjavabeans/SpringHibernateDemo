# SpringHibernateDemo
Spring + Hibernate + Maven 
---------------------------
This project is demo project of Spring IoC and Hibernate.

-----------------------
Technology Used 
-----------------------
Spring 4
Hibernate 4
Oracle 11g
Maven 4

-----------------------
Installation Steps
-----------------------
1. Build the project
2. Execute the below query -

CREATE TABLE EMPLOYEE 
   (EMP_ID NUMBER, 
	EMP_NAME VARCHAR2(2000 BYTE)
 );

 3. Update db.properties file
